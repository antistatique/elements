Text decoration

The text-decoration-line property sets the kind of decoration that is used on text in an element, such as an underline or overline.

Set the decoration of a text using the text-decoration-underline, text-decoration-overline, text-decoration-line-through, and text-decoration-none utilities.

Opacity

The opacity property sets the opacity level for an element. The opacity level describes the transparency level, where 1 is not transparent at all, .5 is 50% visible, and 0 is completely transparent.

Set the opacity of an element using the opacity-{value} utilities.
